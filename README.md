## About

Official discussion forum for the SPROuT course at Charles University. This space facilitates communication, collaboration, and knowledge sharing among students and instructors throughout the 5-day course.

**Access the Forum**: [GitHub Discussions](https://github.com/orgs/CUNI-SPROuT/discussions)

## How to Participate

### Getting Started

1. **Access the forum** via the Discussions tab in any SPROuT repository or the organization page
2. **Browse existing discussions** to see what others are talking about
3. **Search before posting** to avoid duplicates
4. **Choose the right category** for your post (see below)

### Posting Guidelines

- **Be respectful and constructive** in all interactions
- **Use clear, descriptive titles** for your posts
- **Provide context** - mention which day/exercise you're referring to
- **Share code using code blocks** with syntax highlighting
- **Mark solutions** when your question is answered
- **Upvote helpful responses** to highlight good answers
